## Implementation

### Bước 1: Tạo script generate-missing-media.ts

File: `scripts/generate-missing-media.ts`

Script standalone dùng Evolink API (`https://api.evolink.ai/v1/images/generations`) để gen 8 ảnh:
- Đọc `COCOLINK_API_KEY` từ `.env`
- POST `{ model: "gpt-image-2", prompt }` cho từng file
- Download response `data[0].url` → save vào `public/<filename>`
- Nếu Evolink fail, fallback sang Pollinations AI (`image.pollinations.ai`)
- Log progress và kết quả

### Bước 2: Chạy script

```bash
bun scripts/generate-missing-media.ts
```
